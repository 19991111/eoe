# 2026-03-12 神经演化实验日志

## 可枯竭能量源实验

### 问题背景
用户指出能量源是无限静态的，Agent可以"躺平"在源上。需要动态、可枯竭的能量源来迫使Agent演化迁徙能力。

### 解决方案
修改 `EnergyFieldGPU` 类:
1. 增加容量追踪 `capacity`, `max_capacity`
2. 脉冲式注入 `_inject_energy_pulse()`
3. 枯竭检测与迁移 `_check_and_respawn()`

### 参数调整
```python
n_sources = 5           # 5个能量源
source_strength = 80    # 脉冲强度
source_capacity = 300   # 减少容量(更快枯竭)
respawn_threshold = 0.15 # 15%枯竭阈值
```

### 实验结果
- Gen 1: 337/500存活 (67%)，能量枯竭明显 (Max E: 180→137)
- Gen 25: 500/500存活 (100%)，稳定
- Gen 40: 496/500存活

### 结论
动态能量源成功迫使Agent演化出:
1. 能量梯度感知
2. 长途奔袭能力
3. 抢夺新资源策略
