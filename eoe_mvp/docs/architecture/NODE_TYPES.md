# EOE Agent 节点类型与演化规则

> 系统架构总结 | 2026-03-13

---

## 一、节点类型 (NodeType)

### 1. 物理接口层 (连接数字大脑 ↔ 物理世界)

| 类型 | 说明 | 作用 |
|------|------|------|
| **SENSOR** | 通用传感器 | 感知环境场 (能量、动能、阻抗、危险) |
| **ACTUATOR** | 通用执行器 | 输出动作 (推进、穿透、防御) |

### 2. 图灵完备核心 (神经网络计算)

| 类型 | 说明 | 作用 |
|------|------|------|
| **ADD** | 加法算子 | 线性组合 inputs |
| **MULTIPLY** | 乘法算子 | 交互项、非线性乘积 |
| **CONSTANT** | 常数节点 | 偏置项 |
| **DELAY** | 延迟算子 | 时序记忆、RNN核心 |
| **THRESHOLD** | 阈值算子 | 非线性激活函数 |

### 3. 演化涌现 (系统自动生成)

| 类型 | 说明 | 作用 |
|------|------|------|
| **MACRO** | 宏算子 | 多节点组合封装 |
| **SUPERNODE** | 超级节点 | 演化棘轮产物，代谢折扣 |

### 4. 遗留传感器 (兼容v0.0)

| 类型 | 说明 |
|------|------|
| SENSE_EPF_CENTER | 能量场中心值 |
| SENSE_EPF_GRAD_X/Y | 能量梯度 |
| SENSE_KIF_CENTER | 动能场中心值 |
| SENSE_ISF_CENTER | 阻抗场中心值 |
| SENSE_DANGER_CENTER | 危险场中心值 |
| LIGHT_SENSOR | 光传感器 |
| GPS_SENSOR | 位置传感器 |
| COMPASS_SENSOR | 指南针传感器 |

## 二、演化规则

### 1. 变异操作 (分裂时触发)

```
父代基因 ──(突变率10%)──> 子代基因
```

**变异类型及概率**:

| 操作 | 概率 | 效果 |
|------|------|------|
| 权重扰动 | 60% | 权重 ±20% 高斯噪声 |
| 添加节点 | 10% | 在两节点间插入新计算节点 |
| 添加边 | 20% | 新建节点间连接 |
| 塑性变异 | 10% | 开启/关闭突触可塑性 |

### 2. 选择压 (唯一标准: 能量)

```
能量 > 阈值 ──> 分裂
能量 < 0    ──> 死亡
```

- **分裂条件**: `energy > REPRODUCTION_THRESHOLD (60.0)`
- **死亡条件**: `energy <= 0`
- **无人工适应度函数**

### 3. 代谢计算

```
基础代谢 = node_count × BASE_METABOLISM
运动代谢 = velocity × ACTIVATION_COST
阻抗代谢 = impedance × multiplier
年龄惩罚 = 1 + α × age²
超级节点折扣 = node_count - supernodes × 0.5
```

### 4. 演化棘轮 (Supernode)

**触发条件**:
- 子代诞生时 5% 概率获得新超级节点

**代谢红利**:
```
有效节点数 = 实际节点 - supernodes × 0.5
```

## 三、大脑结构

```
输入层 (Sensors) 
    │
    ▼
──── 隐层 (ADD/MULTIPLY/THRESHOLD/DELAY) ──── 循环连接 (DELAY)
    │
    ▼
输出层 (Actuators)
```

**节点数量演化**:
- 基线: 3-4 节点
- 迷宫压力: 7 节点 (空间记忆)
- 演化棘轮: 8-9 节点 (模块化)
- 当前极限: **9 节点** (23.3%)

## 四、关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| MUTATION_RATE | 0.1 | 基础突变概率 |
| BASE_METABOLISM | 0.01 | 每节点基础代谢 |
| ACTIVATION_COST | 0.1 | 运动代谢系数 |
| AGE_ALPHA | 0.00001 | 年龄惩罚系数 |
| SUPERNODE_BONUS | 0.5 | 超级节点代谢折扣 |
| REPRO_THRESHOLD | 60.0 | 分裂能量阈值 |
