## 一、系统概述

EOE (Emergent Organic Engine) 是一个连续时间的人工生命演化系统，以**能量**为唯一选择压，不使用人工适应度函数。

### 核心指标

| 指标 | 数值 |
|------|------|
| 演化步数 | 10,000+ |
| 最高节点数 | **9节点** (23.3%) |
| 平均节点数 | 7.87 |
| 超级节点 | 1.73/agent |
| GPU加速 | 210x |

## 二、已激活机制

### 1. 黑暗森林 (Dark Forest)
- **机制**: 捕食者-猎物军备竞赛
- **参数**: `predation_range=3.0`, `mutation=0.2`
- **效果**: 催生更高复杂度的大脑

### 2. 迷宫与绿洲 (Labyrinth & Oases)
- **机制**: 
  - 高阻抗墙壁 (`impedance=50.0`) - 墙壁区域代谢指数增长
  - 能量集中刷新 - 只有绿洲位置有能量
- **参数**: `maze_density=0.25`, `oasis_count=5`
- **效果**: 强制空间记忆，7节点突破

### 3. 演化棘轮 (Supernode Ratchet)
- **机制**: 常见模式自动折叠为SUPERNODE
- **参数**: `metabolic_bonus=0.5` (5折代谢!)
- **效果**: 
  - 8节点大脑出现 (43.3%)
  - 有效节点数 = 实际节点 - supernodes × 0.5

### 4. 凛冬将至 (Seasons)
- **机制**: 季节性能量倍率
- **参数**: `winter_multiplier=0.1` (冬季仅10%能量)
- **效果**: 极寒筛选，9节点突破

## 五、结论

通过**环境压力** + **代谢激励**的双重驱动，系统成功突破了7-9节点的复杂度瓶颈。演化棘轮机制解决了"代谢天花板"问题，使无限复杂度成为可能。
