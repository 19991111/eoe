# EOE 项目问题描述

## 一、项目背景

EOE (Evolving Organisms Ecosystem) 是一个自主进化的多智能体系统，旨在模拟自然界中复杂生命结构的涌现过程。系统基于神经演化（NeuroEvolution）框架，通过遗传算法+Hebbian学习驱动Agent的大脑（神经网络）自主演化。

### 核心目标
探索如何通过演化机制使简单生物涌现出复杂的认知结构（感知→记忆→决策→行动）。

---

## 二、系统架构

### 2.1 个体结构
- **节点类型**: SENSOR（感知）、PROCESSOR（处理，含ADD/MULTIPLY/THRESHOLD/DELAY）、ACTUATOR（行动）
- **拓扑**: 有向无环图（DAG），支持反馈边
- **学习**: Hebbian学习（突触可塑性）+ SuperNode（结构复用）

### 2.2 环境
- **能量场**: 动态高斯斑块，移动式分布
- **压力机制**: 渐进式能量衰减（每3000步衰减5%）
- **奖励**: 仅top 1%个体获得能量奖励

### 2.3 演化机制
| 机制 | 描述 |
|------|------|
| Morphology | 节点/边的增删 |
| Ontogeny | 生长与分裂 |
| Stigmergy | 协作信号 |
| Thermal | 环境适应 |
| Red Queen | 熵增驱动 |

## 三、核心问题：极简结构局部最优

### 3.1 问题现象

**经过15000步演化后，种群呈现以下特征：**

| 指标 | 结果 |
|------|------|
| 存活率 | 100% (10000/10000) |
| 平均节点数 | 4.3 |
| 最优结构 | **2节点** (SENSOR→ACTUATOR) |
| 节点分布 | 2节点最多，4节点次之，>6节点极少 |

**冠军基因组分析：**
```
# Top 1 能量冠军
节点: 2 (SENSOR → ACTUATOR)
边: 3
能量: 153520.5
```

### 3.2 问题本质

**简单结构持续压制复杂结构，即使在以下条件下：**

1. ✅ 动态能量场（斑块移动，迫使追踪行为）
2. ✅ 渐进式压力（能量衰减，模拟资源稀缺）
3. ✅ Hebbian学习（91%个体活跃学习）
4. ✅ SuperNode机制（10个模式已冻结）
5. ✅ 能量上限补丁（节点越多，上限越高）

**但简单结构仍然全面胜出。**

## 四、已尝试的解决方案

### 4.1 能量上限补丁

**设计逻辑：**
- 能量上限 = 基础值(1000) + 节点数 × 150
- 3节点上限: 1450, 5节点上限: 1750, 10节点上限: 2500
- 吃饱后速度降低（penalty_factor=0.8）

**实验结果：**
| Step | 平均能量 | 能量上限 | 效果 |
|------|---------|---------|------|
| 500 | 0.9 | 1450 | 无饱腹个体 |
| 7000 | 6.3 | 1450 | 无饱腹个体 |
| 15000 | 11.4 | 1450 | 无饱腹个体 |

**结论：能量上限未能驱动复杂化** - 个体平均能量远低于上限，补丁未产生选择压力。

### 4.2 其他尝试

| 方案 | 结果 |
|------|------|
| 强制复杂初始化（min_nodes=5） | 存活率下降，需手动调整代谢参数 |
| 更高能量场强度 | 简单结构仍为主流 |
| 降低代谢成本 | 仅延迟死亡，未改变结构分布 |

## 六、深度分析

### 6.1 简单结构优势来源

| 因素 | 简单结构(2节点) | 复杂结构(6节点) |
|------|----------------|-----------------|
| 基础代谢 | 0.4 | 0.4 × 6 = 2.4 |
| 运动成本 | 0.05 | 0.05 × 6 = 0.3 |
| 推理成本 | 低 | 高 |
| 能量获取 | 相同 | 相同 |
| **净收益** | **高** | **低** |

**核心矛盾：收益不随复杂度增加而增加，但成本线性增加。**

### 6.2 能量场与行为

观察日志显示：
- 斑块位置: (64,81), (69,32), (37,78) 等
- 个体能够追踪斑块
- 但**简单的感知-直接行动**已足够生存

### 6.3 学习vs演化

- Hebbian学习: 72-99%个体活跃
- 但学习只优化权重，不改变拓扑
- 拓扑演化（增删节点）可能存在瓶颈

## 七、待专家分析的问题

### Q1: 收益-成本不对称
**问题**: 当前环境中，复杂结构的收益（更准确的感知、更复杂的决策）无法覆盖其额外成本。如何设计环境使得复杂结构获得净收益优势？

### Q2: 能量获取机制
**问题**: 当前所有个体从能量场获取的能量相同（基于位置采样）。是否需要引入"认知溢价"——复杂个体能获取更多信息，从而获取更多能量？

### Q3: 代谢模型
**问题**: 当前代谢是线性叠加（节点数 × 基础代谢）。是否有更符合生物学的模型（如非线性、层级式）？

### Q4: 演化驱动力
**问题**: 在静态/动态能量场中，简单反射已足够。如何引入真正需要"记忆"和"推理"的任务？

### Q5: 涌现的触发条件
**问题**: 从寒武纪生命大爆发看，复杂结构的涌现需要特定条件。我们的系统缺少什么？
