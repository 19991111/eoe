# 2026-03-15 工作日志

## EOE项目 - 关键Bug修复

### 问题发现
- **现象**: v16.8实验中Agent完全静止（速度=0）
- **影响**: 所有Agent不移动，在可见期饿死

### 根因
`batched_agents.py`中`step()`方法的bug：
```python
# 原代码：当brain_fn=None时，输出全零！
if brain_fn is not None:
    brain_outputs = brain_fn(batch)
else:
    brain_outputs = torch.zeros(batch.n, n_outputs, device=self.device)
```

### 修复方案
```python
# 修复后：默认调用forward_brains
elif self.brain_matrix is not None and env is not None:
    sensors = self.get_sensors(env)
    brain_outputs = self.forward_brains(sensors)
```

### 验证结果
- 使用随机脑输出测试：Agent能够移动（位移~0.7/20步）
- 随机初始化的大脑矩阵输出接近零是正常的（稀疏随机图）
- 演化会逐渐产生有效的大脑连接
