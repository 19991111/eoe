## 下午发现：能量消耗过高问题

### 症状
- 预实验 Agent 100% 死亡，夏天都活不过去
- 每步消耗 7-8 能量，30步就死光

### 根因分析
1. **冠军脑结构**: 37节点 + 61边 + 5个PORT节点
2. **基础代谢**: 37×0.02 + 61×0.02 = 1.96/步
3. **PORT能耗** (二次方):
   - port_motion: 0.3 × motion²
   - port_offense: 0.5 × offense²
   - port_defense: 0.4 × defense²
   - port_repair: 0.6 × repair²
4. **总消耗**: ~7-8/步（远超预期的1.96）

### 解决方案
预实验需要降低代谢参数：
- metabolic_alpha: 0.02 → 0.01
- metabolic_beta: 0.02 → 0.01

这样基础代谢降到 ~1/步，加上 PORT 成本 ~2-3/步，总共 ~3-4/步。
