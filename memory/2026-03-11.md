### 3. LLM Demiurge 系统

**llm_demiurge_loop.py**
- `BASE_CONSTITUTION`: 不可变宪法，定义LLM角色和红线
- `generate_epoch_report()`: 四大维度战报生成器
  - 维度1: 生存与死亡 (饿死/冻死比例)
  - 维度2: 行为里程碑 (碰触/携带/贮粮次数)
  - 维度3: 能量收支 (摄入/消耗/效率)
  - 维度4: 脑容积趋势 (节点数/边数/复杂度)
- DeepSeek API 集成 + JSON Schema 校验
- 飞书通知

**daemon_runner.py**
- GPU 自动检测 (支持单GPU回退)
- 每100代纪元阻断 + LLM 介入
- 检查点管理 (champions/目录)
- 容错与断点续传
- demiurge_decisions.log 日志
